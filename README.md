<b>TRIFENIX<b> 

Instalación: </p>
- Clonar proyecto </p>
- Ingresar a carpeta del proyecto </p>
cd en-construccion </p>
- Instalar módulos de Node.js </p>
npm install </p>
- Instalar Serve </p>
npm install serve -g </p>
- Construir el sitio </p>
serve build </p>
- Ingresar al sitio por el puerto 5000 </p>
http://localhost:5000
