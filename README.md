<b>TRIFENIX<b> 

Instalación: </p>
- Clonar proyecto </p>
- Ingresar a carpeta del proyecto </p>
cd en-construccion </p>
- Instalar módulos de Node.js </p>
npm install </p>
- Iniciar Serve, si no lo tienes instalalo </p>
npm install serve </p>
- Iniciar el sitio </p>
serve -s build </p>
- Ingresar al sitio por el puerto 5000 </p>
http://localhost:5000
